# Rule-Based Expert System

## Project Description
This project is a Rule-Based Expert System implemented using Python. It accepts user symptoms (facts) and infers conclusions using forward chaining.

## Features
- Rule engine using if-then rules
- Facts base
- Forward chaining inference
- Multi-step reasoning
- Inference logging

## Technologies Used
- Python
- Google Colab

## Example Input
fever, cough

## Example Output
Rule Applied: ('fever', 'cough') -> flu
Final Conclusion: ['fever', 'cough', 'flu']
